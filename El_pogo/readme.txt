Bonjour,


Il suffit d’entrer dans le terminal, d’entrer dans le fichier 
et de taper la commander make.
Une fois le "make" effectué 
Le nom de l'exécutable est : pogo

Une fois dans le programme si vous entrez autre chose qu'un nombre entier l'exécution s'arrêtera, il faudra le redémarrer.
Le programme fonctionne en cycles de 10000 tours (1min) et vous pourrez choisir de continuer ou non.

Attention le save.pogo créé pendant le programme semble être trop lourd à afficher pour les ordinateurs de l'école. 
Si on veut l'ouvrir il vaudrait mieux faire une simulation courte avec un nombre limité de danseurs.
Supprimez le fichier après chaque simulation si il s'avère trop long.

Sur ce, nous vous souhaitons un agréable pogo.
